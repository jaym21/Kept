# Kept

Kept is a modern, privacy-focused personal finance app designed to help users organize, analyze, and understand their entire financial life in one place.
Built with Kotlin, Jetpack Compose, Clean Architecture, and Room, Kept aims to be a reliable offline-first companion for tracking:

- Equity & stock portfolios
- Insurance policies
- Mutual funds
- Fixed deposits & recurring deposits
- Real estate assets
- Gold investments
- Upcoming premiums, SIPs, EMIs, and dues
